## Development

clone this repository

```bash
git clone git@github.com:imc-ux/AdvancedSvelte.git

# npm
npm install

# Development Server

Start the development server on http://127.0.0.1:7070/

# server run

npm run server 

```

## Production

Build the application for production:

```bash
npm run build
```
