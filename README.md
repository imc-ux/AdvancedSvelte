## Setup

Make sure to install the dependencies:

```bash
# yarn
yarn install

# npm
npm install

# pnpm
pnpm install
```

## Development Server

Start the development server on http://127.0.0.1:7070/

```bash
npm run server
```

## Production

Build the application for production:

```bash
npm run build
```
